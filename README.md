多引擎快捷搜索工具

一个强大的用户脚本，为搜索引擎页面添加可定制的多引擎快捷搜索栏，支持拖拽排序、智能识别和键盘快捷操作。

🚀 核心功能

🔍 智能搜索集成

· 多引擎支持: 在任意搜索引擎页面底部显示快捷搜索栏
· 关键词同步: 自动提取当前页面的搜索关键词
· 一键跳转: 点击任意引擎按钮使用对应搜索引擎搜索当前关键词
· URL 直连: 支持直接输入网址进行跳转

🎨 个性化定制

· 拖拽排序: 通过拖拽调整搜索引擎按钮顺序
· 自定义引擎: 手动添加或自动识别新搜索引擎
· 图标定制: 支持 SVG、图片、文字、表情符号等多种图标形式
· 视觉主题: 现代化的毛玻璃设计风格

⌨️ 高效操作

· 键盘快捷键:
  · Alt + S: 打开搜索遮罩层
  · Alt + E: 打开引擎管理面板
  · Alt + M: 打开/关闭汉堡菜单
  · ESC: 关闭当前弹窗
· 触摸优化: 专门为移动设备优化的触摸交互，滑动引擎栏不会导致输入框失焦

⚙️ 智能管理

· 自动识别: 智能识别当前页面的搜索引擎配置
· 批量管理: 可视化引擎启用/禁用管理
· 配置导出: 自动保存所有个性化设置
· 重置恢复: 一键恢复默认搜索引擎配置

🛠️ 技术特性

架构设计

· 模块化架构: 功能模块清晰分离，便于维护和扩展
· 状态管理: 统一的全局状态管理，确保数据一致性
· 事件防抖: 智能的事件防抖处理，提升性能表现

用户体验

· 响应式设计: 自适应不同屏幕尺寸
· 无障碍支持: 完整的 ARIA 标签和键盘导航支持
· 流畅动画: 细腻的过渡动画效果
· 智能隐藏: 滚动时自动隐藏，需要时智能显示

兼容性

· 多浏览器支持: 兼容主流用户脚本管理器
· 多引擎适配: 支持 Google、Bing、百度、DuckDuckGo 等主流搜索引擎
· 移动端优化: 完善的触摸事件处理

搜索研究者

效率追求者

💡 特色功能详解

智能关键词提取

· 从 URL 参数自动解析搜索关键词
· 监控页面输入框实时同步内容
· 支持百度等特殊搜索引擎的输入框识别

引擎自动识别

· 分析页面表单结构识别搜索引擎
· 从 meta 标签提取站点信息
· 支持常见搜索引擎模式匹配

触摸交互优化

· 引擎栏触摸操作不会干扰输入框焦点
· 智能的滚动隐藏/显示逻辑
· 流畅的拖拽排序体验

这个脚本将多个搜索引擎整合到一个便捷的浮动工具栏中，极大提升了搜索效率和用户体验，是搜索重度用户的必备工具。
